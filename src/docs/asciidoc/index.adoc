==== 요금 정책 설명

- 노선 추가 요금이 존재할 수 있음
- 청소년(13세 이상 19세 미만) : 350원 공제 후 20% 할인
- 어린이(6세 이상 13세 미만) : 350원 공제 후 50% 할인
- 10 ~ 50 : 5Km당 100원의 추가 요금
- 50 ~ : 8km당 100원의 추가 요금
